Añade acción a la navegación de tu sitio web

Librería que permite añadir una nave espacial a cualquier sitio web, con la cual puedes moverte por toda la página y disparar sobre los enlaces y botones para visitarlos

Ver en GitHub

Descargar, copiar y a destruir!

  • Descargate los archivos js y la carpeta de asset
  • Añade el siguiente código en su sitio web
    document.addEventListener('DOMContentLoaded', function () {
        var arcadeNavigator = new ArcadeNavigation();
        arcadeNavigator.init();
    });

Configuración

Puedes un objeto de Configuración cuando inicias el juego

Estos son los datos de configuración que tiene por defecto

Importante! La ruta es la carpeta donde están alojadas las imagenes en el servidor 

    new ArcadeNavigation({
        id: 'contentGameArcadeNavigation',
        zIndex: 3,
        imagenes: {
            nave: 'player.png',
            laser: 'bullet.png',
            explosion: 'explode.png',
            hueco: 'hole.png',
            particula: 'particle.png'
        },
        ruta: '/assets/',
        delay: 400,
        enemigos: [
        'a', 'button', '[role=button]',
        'input[type="submit"]', 'input[type="reset"]'
        ]
    });

La nave aparece en todas las páginas donde añadas el script

Caracteristicas

  • Scroll: El scoll de la página se desplazará con la nave, permiento ver la página al completo
  • No interfiere: Aunque el juego aparezca encima del contenido puedes seguir utilizando el ratón sin problemas

Personalización

Sientete libre cambiar o editar la imagene de la nave o el laser para que se adapte más a tu estilo, recuerda reemplazarlas en la carpeta de asset o cambiando el nombre en las optiones
